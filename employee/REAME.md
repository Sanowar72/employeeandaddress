* Springboot

* lastName
* address

## Address attributes
* addressId
* street
* city
* state
* zip code
* employee

* Address Controller

## Service
* EmployeeService
* AddressService

## Model

* Address

## Repository
* EmployeeRepository
* AddressRepository
* Used CrudRepository and Custom Query method
